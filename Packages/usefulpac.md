# Installed packages

## basis

```shell
sudo pacman -S yay
sudo pacman -S xsel
sudo pacman -S pasystray
sudo pacman -S base-devel
sudo pacman -S lxappearance
```

## latex

```shell
sudo pacman -S texlive-most texlive-lang biber
sudo pacman -S pandoc
```

## image and PDF

sudo pacman -S ds9

sudo pacman -S zathura-pdf-poppler zathura-ps zathura-djvu
